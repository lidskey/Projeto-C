Funcionamento do sistema de cadastro

1. Para iniciar o cadastro, o funcionário deve fazer o login com seu usuário (admin) e senha (123) para ter cesso, caso contrário não será possível acessar.
Primeiramente deve-se preencher o login, apertar enter, preencher a senha e então apertar o enter novamente.
2. Caso as informações de login estiverem erradas, não será possível prosseguir com o acesso ao cadastro. O sistema informará que o login está incorreto e o sistema solicitará para que o usuário tente novamente.
3. Após as informações de login, o usuário poderá escolher entre CADASTRAR (1) e SAIR (0). Escolhendo a opção 0, o sistema finalizará a execução.
4. Caso, escolha a opção 1 (CADASTRAR) o funcionário tem acesso as telas de preenchimento dos dados. O sistema pede as informações que precisa para realizar o cadastro (nome, cpf, email, data de nascimento, telefone, rua, complemento, bairro, cidade, estado, cep, data de diagnostico, comorbidade, se tiver comorbidade, qual comorbidade).
5. Quando preenchida a data de nascimento o sistema calculará a data do paciente para classificar se está no grupo de risco por idade.
6. Após o preenchimento dos dados, o sistema salvará os dados do paciente em um arquivo .txt chamado "pacientetodos.txt" e será gerado na mesma pasta do executável.
